# Test Cases Example

## Test Case 1

Test Case ID: TC001  
Title: Login with valid data  
Description: Verify that user can login with correct email and password.

Steps:
1. Open login page
2. Enter valid email
3. Enter correct password
4. Click login button

Expected Result:
User successfully logs in and dashboard page opens.


## Test Case 2

Test Case ID: TC002  
Title: Login with incorrect password  
Description: Verify error message when wrong password is entered.

Steps:
1. Open login page
2. Enter valid email
3. Enter wrong password
4. Click login button

Expected Result:
Error message should appear.


## Test Case 3

Test Case ID: TC003  
Title: Empty login fields  
Description: Verify validation when fields are empty.

Steps:
1. Open login page
2. Leave fields empty
3. Click login button

Expected Result:
Validation message should appear.
