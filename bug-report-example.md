# Bug Report Example

## Bug Title
Login button does not respond after entering valid credentials

## Bug ID
BUG-001

## Module
Login Page

## Environment
- Device: Laptop/Desktop
- OS: Windows 11
- Browser: Chrome
- Version: Latest available version

## Severity
High

## Priority
Medium

## Steps to Reproduce
1. Open the login page
2. Enter a valid email address
3. Enter the correct password
4. Click the login button

## Expected Result
The user should be logged in successfully and redirected to the dashboard.

## Actual Result
Nothing happens after clicking the login button. The user stays on the login page.

## Additional Notes
This issue prevents users from accessing the application even when valid credentials are entered.
